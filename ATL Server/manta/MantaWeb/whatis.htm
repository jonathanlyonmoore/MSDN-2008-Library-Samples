Welcome to the MantaWeb Sample!

MantaWeb is an ATL Server sample which demonstrates how ATL Server can be used to create a web-based solution.  This sample uses such ATL Server features as: stenciled pages to separate static and dynamic content, attributes which simplify the development process, SMTP classes for sending e-mail messages, and ATL Server's validation and exchange support.

MantaWeb itself is designed to be an online portal for sending and receiving e-mail, managing your appointments with an online calendar, keeping track of your current tasks with the task list, and a convenient place to store your most used files so you can access them from any computer in the world equipped with a web browser.
