SRFSyntax Sample: Server Response File Demonstration

Demonstrates how to use attributed code to implement an ATL Server request handler that illustrates advanced features of server response files.

The SRFSyntax sample illustrates how to:

  • Associate replacement methods with replacement tags using the tag_name attribute.

  • Parse replacement tag parameters and handle them in a replacement method.

  • Control the flow of execution within a SRF file using if and while statements.

  • Include one SRF file in another using the include tag.

Security Note

This sample code is provided to illustrate a concept and should not be used in applications or Web sites, as it may not illustrate the safest coding practices.

Requirements

  • IIS

Building and Running the Sample

To build and run this sample

  1. Open the solution file, SRFSyntaxSample.sln, in the Visual Studio development environment.

  2. Build the solution. This will also deploy the solution to the local Web server.

  3. Use a Web browser to view http://localhost/srfsyntaxsample/srfsyntax.srf.
