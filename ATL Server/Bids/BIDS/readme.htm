Building and Running the Bids Online Sample

  1. Open the solution file, Bids.sln, in the Visual Studio development environment.
  2. If your web directory is in a location other than c:\inetpub\wwwroot, modify the data source constants in StdAfx.h accordingly.
  3. Build the solution.  This will also deploy the solution to the local web server, including the bidinfo.mdb database.
  4. Use a web browser to view http://localhost/bids/default.htm  and navigate through the site.

Important Note:

This sample is only provided to demonstrate features of ATL Server and should not be used in a production environment.  For example, this sample is not hardened against malicious input and performs no input validation checking, user sessions are vulnerable to hijacking, the database is stored in a web accessible location with weak permissions, and despite the fact that users may specify a time to end the auction, functionality is not included to actually end the auction.
